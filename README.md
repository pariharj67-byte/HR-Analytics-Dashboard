# 📊 HR Analytics Dashboard (Power BI)

## 🔍 Project Overview

This project focuses on analyzing employee attrition using Power BI.
The dashboard provides insights into workforce trends, salary distribution, and job roles with high attrition.

---

## 📌 Key Insights

* 🔺 Highest attrition in **26–35 age group (116 employees)**
* 💰 Majority attrition from salary **below 5K**
* 🧪 **Laboratory Technician** role most affected
* 👨 Male attrition is significantly higher than female

---

## 📈 Features

* Interactive dashboard with filters
* Attrition analysis by Age, Salary, Job Role & Education
* KPI cards for quick overview
* Tooltip-based insights for better user experience

---

## 🛠 Tools Used

* Power BI
* Data Visualization
* DAX (Basic Measures)

---

---

## 🎯 Conclusion

This dashboard helps HR teams identify key factors behind employee attrition and supports data-driven decision making.
